# 🌐 Job Listings Scraper

A Python web scraper that pulls job postings (title, company, and more) from a sample job board using `requests` and `BeautifulSoup`.

## 📌 Features

- Scrape job titles and companies from a website
- Clean command-line output
- Extendable for additional fields or sites
- Easily saves output to CSV or JSON (optional)
